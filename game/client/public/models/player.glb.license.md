## Work

`player.glb` — low-poly humanoid player mesh in neutral rest pose (static body mesh,
two PBR materials). Intended as the base asset for proportion morph targets in a
later sub-ticket; the live renderer still uses the procedural box fallback until
wired (ticket 161).

## Author and source

- **Author:** Autogame project (Blender-authored original mesh).
- **Source:** Exported from the repository `assets/models` branch (`player.glb`, ~15 KB,
  commit `fdbcccc` — “low-poly GameCube-style hero” v1). Copied into
  `game/client/public/models/` for ticket 185, sub-ticket 02.
- **Not** derived from third-party asset packs (e.g. Quaternius); an in-repo placeholder
  suitable for morph authoring until a rigged pack asset replaces it.
