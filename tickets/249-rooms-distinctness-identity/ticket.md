# 249-rooms-distinctness-identity

## Difficulty: medium

## Goal

Make the room layouts read as distinct biomes (playtest: crowded vs open differ, but all share one grey/green palette).

## Acceptance Criteria

- Per-profile floor/wall materials (open=sandy, crowded=dark metal); 1-2 landmark props per profile; differentiate crowded vs default structurally; hazards/verticality for the empty 'open' profile; doorway markers in large rooms. Visual review.

## Verification

Run the harness checks (vitest server+client) and verify the acceptance criteria above.
